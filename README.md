# YanMo的个人博客

一个基于HTML、CSS和Python的个人博客网站，专注于信息安全领域的内容分享。

## 网站特点

### 视觉设计
- **赛博朋克风格**：深色背景配合霓虹青色和洋红色的发光效果
- **响应式布局**：适配不同屏幕尺寸，包括移动端和桌面端
- **动画效果**：悬停动画、扫描线效果、霓虹发光效果
- **文章封面**：每篇文章都有相关的封面图片

### 功能模块

#### 1. 前端页面
- **首页**：个人简介、最新安全文章、快速导航
- **关于我**：个人信息、技能展示、安全研究经验
- **文章列表**：安全相关文章分类展示，带有封面图片
- **文章详情**：详细的安全技术文章内容
- **联系页面**：联系方式和联系表单
- **留言板**：用户留言功能
- **友链页面**：友情链接展示
- **友链申请**：友链申请表单
- **留言板申请**：留言提交表单

#### 2. 后端功能
- **Python服务器**：处理表单提交和API请求
- **SQLite数据库**：存储留言、友链申请和文章数据
- **API接口**：提供数据访问和管理功能
- **管理员后台**：管理留言、友链和文章

#### 3. 特色功能
- **实时统计**：待审核友链、已发布文章、总访问量的实时统计
- **悬停提示**：QQ和Email联系方式的悬停显示
- **GitHub链接**：点击直接跳转到GitHub个人主页
- **赛博朋克风格**：贯穿整个网站的赛博朋克视觉元素

## 技术栈

- **前端**：HTML5, CSS3, JavaScript
- **后端**：Python 3, SQLite
- **服务器**：Python内置http.server
- **图片生成**：使用Trae API生成文章封面图片

## 如何运行

1. 确保安装了Python 3
2. 进入网站目录
3. 运行服务器：`python server\server.py`
4. 访问网站：http://localhost:3000
